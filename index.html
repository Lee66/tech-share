Vue.js

Vue.js（读音 /vjuː/, 类似于 view）是一个构建数据驱动的 web 界面的库。Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

Vue的定位与React相同，只关注视图层。而Angular的定位则是大而全。

作者：尤雨溪 美籍华人 颜值高

数据绑定

  • 创建Vue实例 
    
	new Vue({
	    el: '#app',
	    data: {},
	})
  • 插入文本：Mustache语法 (双大括号) 
    <span>Message: {{ msg }}</span>

  • 不止是简单值，同时支持全功能的JavaScript表达式 
    {{ number + 1 }}

  • 双 Mustache 标签将数据解析为纯文本而不是 HTML。{{{raw_html}}}：3个大括号可以将数据解析为HTML

  • 绑定属性：v-bind 指令 
    <a v-bind:href="url"></a>

  • v-bind的缩写 : 
    <a :href="url"></a>

条件渲染

  • v-if、v-else 指令 
    <h1 v-if="ok">Yes</h1>
<h1 v-else>No</h1>

  • template v-if 
    <template v-if="ok">
  <h1>Title</h1>
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
</template>

  • v-show 指令 
    <h1 v-show="ok">Hello!</h1>

  • v-if 与 v-show 的区别

    v-show 等同于 display: none, 而v-if不会创建dom元素

列表渲染

  • v-for 指令 
    
<ul>
  <li v-for="item in items">
    {{ item.message }}
  </li>
</ul>

  • $index 特殊变量

  • template v-for 
    
<ul>
	<template v-for="item in items">
		<li>{{ item.msg }}</li>
		<li class="divider"></li>
	</template>
</ul>

  • 对象 v-for

    遍历的是对象值
    多一个特殊变量：$key

事件绑定

  • v-on 指令 
    
	<button v-on:click="shit">shit</button>

  • v-on 指令的缩写 
    
		<button @click="shit">shit</button>

  • 为事件函数传递参数 (内联语句) 
    
	<button v-on:click="shit('bug')">shit</button>

  • 还可以添加为事件添加修饰符，最常用的是按键修饰符： 
    <input v-on:keyup.13="submit">

  • 记住所有的keyCode是比较困难的，所以同时提供了常用的按键别名： 
    enter tab delete esc space up down left right

class、style以及计算属性

同属于绑定属性，只不过特殊一点

  • class、style：可以绑定为一个对象或者数组

  • 计算属性：模板中放入太多逻辑会让模板过重且难以维护
